Bem-vindo

Esta aplicação exemplifica o fluxo de cadastro, login e gerenciamento de tarefas usando JWT.

Cadastro de Usuário Login Perfil Minhas Tarefas

Como usar

1. Cadastre um usuário.

2. Faça login para receber o token JWT.

3. Acesse a página de tarefas para criar e listar as tarefas do seu usuário.

Front-end simples para demonstrar chamadas REST em Node.js + Express + SQLite.
